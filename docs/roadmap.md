# 开发路线

## Phase 1: 骨架

- 前端研究页
- 后端 API
- 统一数据模型
- Demo 报告链路

## Phase 2: 真实数据接入

- 股票基础资料
- 日线/分钟线
- 板块和概念映射
- 公告和监管事件
- 热门新闻聚合

## Phase 3: 核心研究能力

- 关键涨跌日检测
- 炒作脉络时间线
- 板块梯队识别
- 风险引擎
- 可比标的引擎

## Phase 4: 社媒与股吧

- 东方财富股吧抓取
- 同花顺评论聚合
- 雪球讨论抓取
- 情绪分类与一致性检测

## Phase 5: AI Agent 化

- 多轮追问
- 自动补充证据
- 研究结论自检
- 输出投资假设与反例

## Phase 6: 跨端打包

- Tauri Desktop
- Android
- 增量推送/桌面通知
- 离线缓存
